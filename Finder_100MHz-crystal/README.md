### Find the minimum (Vmin) and maximum (Vmax) values of a wave signal at any frequency + 100MHz crystal oscillator

**Board:** `Raspberry Pi Pico`

**Firmware:** `RPI-PICO-20240602-v1.23.0.uf2`

**IDE:** `Android Micro REPL App`

**Input PIN:** `ADC 26`

**Input resistor:** `1K Ohm`

**Oscillator:** `100MHz external crystal oscillator for testing`

**Note:** The frequency of the input signal doesn't affect the finder. Vmin and Vmax should be between 0V and 3.3V. If Vmin equals Vmax, the input signal is not a wave. Therefore, the simplest tool to determine if a testing circuit is an oscillator or not?!
